Project Descriptions

Theatres

First ever attempt at web design implementing Bootstrap. Here is a webpage dedicated to a cinema giant called "Amphitheatre" displaying availabilities for possible showtimes.

The page is responsive and was a real first attempt at properly formatting using HTML and using more complicated forms of CSS.

Connect4

Rediscover the thrill of Connect4 online. The user interface combines simplicity with excitement, ensuring an enjoyable experience for players of all ages.

Gather around for friendly competition as you strive to connect four discs in a row! Whether you triumph horizontally, vertically, or diagonally, the excitement never ends in this iconic game.

Toytower

Toytower is an engaging e-commerce platform dedicated to offering a diverse range of toys catering to various age groups. Built using PHP and leveraging the Fat-Free Framework, it stands as an early project and first use into the MVC architecture.

The platform boasts an array of playful and interactive features, with each product card showcasing quirky and amusing descriptions. Navigating through Toytower is not only convenient but also enjoyable, offering users a delightful shopping experience.

Travel@Ease

Travel@Ease is an online platform for booking tours applying the likes of React and Node.js. Users can register and login where authentication is then applied, giving the user access to either the administrative panel or their user profile.

Administrators possess the ability to create, update, and remove tours listed on the website.

Users can effortlessly navigate through their profile, where they have the freedom to update personal information, browse past bookings, leave reviews and rate their past experiences.

Library Management System

Developed with Spring Boot, here is a simple library management system that uses Spring's easy to use authentication and login to give access to users and administrators.

One of the first projects to apply user authentication, the goal was to allow admins access to all CRUD operations and to restrict common users by implementing access denied pages and redirecting them to view only pages.
